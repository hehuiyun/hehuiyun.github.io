css简单实现stick footer

前言

什么是stick footer?

stick footer就是让footer元素固定在底部，并且满足以下两个要求：

  1. 当内容不足满屏时，footer紧贴最底部；
  2. 满屏需要下拉时，footer就在内容的底部。
