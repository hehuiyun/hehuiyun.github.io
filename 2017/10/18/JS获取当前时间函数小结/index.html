JS获取当前时间函数小结

前言

之前在总结倒计时的时候，发现自己对于获取系统时间这部分有点生疏，所以也一起总结一下。

1.getYear()

作用：获取当前年份（2位）

注意：该函数获取当前年份后两位，但是当年份的第二位数为0时，会返回三位数1+两位数

2.getFullYear()

作用：获取完整的年份

3.getMonth()

作用：获取当月是一年中的第几月，为基于0的值（0表示一年中的第一月）。

参数: 无

返回值: getMonth返回一个0 到 11的整数值： 0 代表一月份，1 代表二月分， 2 代表三月份，依次类推。

4.getDate()

作用：获取当天是一个月中的第几天。

参数: 无

描述: getDate()返回一个1 到 31的整数值。

5.getDay()

作用：获取当天是星期几

参数: 无

描述: getDay()返回一个整数值： 0 代表星期日， 1 代表星期一，2 代表星期二， 依次类推。

6.getTime()

作用：获取当前时间距离1970年1月1日0时0分0秒的毫秒数

参数: 无

描述: getTime 方法的返回值一个数值，表示从1970年1月1日0时0分0秒（UTC，即协调世界时）距离该日期对象所代表时间的毫秒数。

7.getHours()

作用：获取当前小时数

参数: 无

描述: getHours返回一个0 到 23之间的整数值。

8.getMinutes()

作用：获取当前分钟数。

参数: 无

描述: getMinutes返回一个0 到 59的整数值。

9.getSeconds()

作用：获取当前秒数

参数: 无

描述: getMinutes返回一个0 到 59的整数值。

10.getMilliseconds()

作用：获取当前毫秒数

参数: 无

描述: getMinutes返回一个0 到 999的整数值。

11.toLocaleDateString()

作用：获取日期（只包含年月日）

12.toLocaleTimeString()

作用：获取时间（只包含上午/下午 时分秒）

13.toLocaleString( )

作用：获取日期和时间（含年月日 上午/下午 时分秒）
